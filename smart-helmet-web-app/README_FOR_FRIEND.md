# 🪖 Smart Helmet Detection System - Dashboard

## ⚡ QUICK START (Sabse Easy Tarika)

### Option 1: SIRF EK FILE OPEN KARO (No Install Required!)
1. `SmartHelmetDashboard.html` file pe **double-click** karo
2. Browser mein open ho jayega — BAS! 🎉

> Ye single file mein poora dashboard hai — koi software install karne ki zaroorat NAHI hai.

---

### Option 2: Full React App Run Karna Ho (Dev Mode)

#### Step 1: Node.js Install Karo
- Download: https://nodejs.org (LTS version)
- Install karo (Next Next karte jao)

#### Step 2: Terminal / CMD Kholo
- `smart-helmet-web-app\frontend` folder mein jao
- Ya phir folder pe right-click → "Open in Terminal"

#### Step 3: Commands Run Karo
```bash
npm install
npm run dev
```

#### Step 4: Browser mein kholo
- http://localhost:5173/

---

### Option 3: One-Click Start (Backend + Frontend dono)
- `start.bat` pe double-click karo
- Ye dono server start kar dega automatically
- (Python + Node.js dono installed hone chahiye)

---
